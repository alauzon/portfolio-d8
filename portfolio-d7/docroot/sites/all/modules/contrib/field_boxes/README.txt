the field_boxes module adds a new box type  calle custom field box.  
This type allow for any field on any entity to be displayed as a box
one must pick an entity type
then select the entity to use (if the entity defines a label, autocomplete is avaiable)
one can also select the current entity (currently this only works for node and taxonomy term)
then select the field to display

one can also select the display mode from which the display settings will be pulled
